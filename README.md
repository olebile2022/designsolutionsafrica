# Graphic Design, Printing & Packaging Website

This is a multi-page static website for a design studio offering graphic design, printing, and packaging services. Built with HTML, CSS, and JavaScript.

## Structure

- `index.html`: Home page with hero section and introduction
- `portfolio.html`: Portfolio page showcasing work
- `services.html`: Services page detailing offerings
- `about.html`: About page with company information
- `contact.html`: Contact page with form and details
- `css/styles.css`: Stylesheet for layout and design
- `js/script.js`: JavaScript for interactivity
- `images/`: Folder for images

## Features

- Multi-page navigation
- Responsive design
- Portfolio grid
- Services showcase
- Contact form with validation
- Scroll animations
- Mobile-friendly

## Setup

1. Add images to the `images/` folder:
   - `hero-bg.jpg`: Background for home page
   - `design1.jpg` to `design4.jpg`: Portfolio images
   - `print1.jpg`, `package1.jpg`: Additional portfolio images
   - `corrugated-box.jpg`, `paperboard-box.jpg`, `rigid-gift-box.jpg`: Packaging product images
   - `graphic-design.jpg`, `printing.jpg`, `packaging.jpg`: Service icons

2. Open `index.html` in a web browser to view the website.

## Customization

- Edit individual HTML files to change content
- Modify `css/styles.css` for styling changes
- Update `js/script.js` for additional interactivity
- Add more pages or sections as needed

## Notes

- The website uses placeholder image paths. Replace with actual images.
- For production, consider adding a backend for form submission or using a static site generator.
